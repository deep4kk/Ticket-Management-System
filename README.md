# Vendor/Customer Complaint & Ticket Management System

A comprehensive Google Apps Script system for managing support tickets with Gmail parsing, SLA engine, and spike detection.

## Features

- **Ticket Creation**: Auto-create tickets from Gmail
- **SLA Engine**: Track and enforce response times
- **Spike Detection**: Alert on recurring issues
- **Conversation Threads**: Track all ticket communications
- **Priority Management**: Auto-categorize and prioritize tickets

## Setup

1. Create Google Sheet with: Users, Tickets, TicketReplies, CategoryRules, AuditLog
2. Set up Gmail label monitoring (optional)
3. Deploy as Web App
